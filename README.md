# EXIF Renamer

A PowerShell GUI script to rename files using EXIF data.

## 

## Authors

* [@remusrigo](https://github.com/RemusRigo)
* [@remusrigo](https://sourceforge.net/u/remusrigo/profile)

## 

## Installation

Just right click on "EXIF Renamer.ps1" and select "Run with PowerShell" or run the "EXIF Renamer.bat"
